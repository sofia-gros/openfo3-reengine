# feat_multi-bsa: 複数 BSA アーカイブ対応

## 概要
Megaton.cs で Fallout - Meshes.bsa / Fallout - Textures.bsa のみ読み込んでいたのを、
config.json の BSA セクションに記載された全アーカイブを読み込むように拡張。

## 変更内容

### GamePaths.cs
- `_bsaFilePaths` フィールドを追加
- `Initialize()` 内で config.json の "BSA" セクションを読み取り、
  Data ディレクトリ下の BSA ファイルのフルパスを収集
- `GetBSAFilePaths()` 公開メソッドを追加

### Megaton.cs
- `_meshFiles` / `_textureFiles` を廃止
- `_meshBsaList: List<(BSAReader, List<BSAFile>)>` を追加
  （メッシュ関連 BSA ごとのファイル一覧）
- `_textureBsaList` を同様に追加（テクスチャ/ Misc BSA 向け）
- `_Ready()` で GamePaths.GetBSAFilePaths() から取得した全 BSA を
  名前判定（mesh/texture/misc 含む）で適切なリストに振り分け
- `EnsureNifParsed()`: `_meshBsaList` を走査して該当 NIF を
  含む BSA から読み込む
- `LoadTexture()`: `_textureBsaList` を走査して該当 DDS を
  含む BSA から読み込む

### BSAReader.cs
- `FilePath` プロパティを追加（コンストラクタで設定）

## 分類ルール
- ファイル名に "mesh" を含む → meshBsaList
- ファイル名に "texture" を含む → textureBsaList
- ファイル名に "misc" を含む → 両方に追加
  （Misc.bsa はメッシュとテクスチャの両方を含むため）

## 確認
- 0 errors, 0 warnings
- "Fallout - Misc.bsa" に含まれるメッシュ/テクスチャが
  解決可能になった
