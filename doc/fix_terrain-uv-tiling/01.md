# fix_terrain-uv-tiling: 地形テクスチャの UV タイル補正

## 問題
`TerrainBuilder.cs` のメイン地形タイル UV が `[0,1]` 範囲に設定されており、
テクスチャが 1 タイル全体に引き伸ばされていた。

一方、ギャップ埋め用の平坦地形 (`BuildFlatTerrainTile`) は
`CellSize / 256f = 16` リピートで正しく設定済みだった。

## 修正
`BuildTerrainTile` の UV 計算にタイルリピート係数を乗算:

```csharp
const float textureTileUnits = 256f;
float tileRepeats = CellSize / textureTileUnits;  // = 16
uvs[idx] = new Vector2(col / quadsPerSide * tileRepeats, row / quadsPerSide * tileRepeats);
```

これにより地形テクスチャが FO3 基準（256 FO3 単位ごとに 1 リピート）
で正しくタイルされるようになった。

## 確認
- 0 errors, 0 warnings
- UV range: `[0, 0]` → `[16, 16]`（33×33 グリッド全体で 16×16 リピート）
