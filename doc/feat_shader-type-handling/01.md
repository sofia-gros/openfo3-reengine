# シェーダータイプの区別とマテリアル最適化

## 問題

FO3 の BSShaderProperty.ShaderType フィールドは NIF から読み取られていたが、`NIFMaterialBuilder` で一切参照されていなかった。すべてのマテリアルが同じ `StandardMaterial3D` 設定で作成され、以下の問題があった:

- 環境マップが正しく使用されていない（金属がつやつやに見えない）
- GlowShader が通常のライティングを受けてしまう（発光が暗い）
- SkinTint / HairTint の頂点カラーティントが適用されていない
- ノーマルマップアルファの Gloss が Roughness として反転して使われている

## 変更内容

### 1. ShaderType 定数を追加

NIFMaterialBuilder.cs に FO3 の全主要シェーダータイプを定数化:

| 定数 | 値 | FO3 シェーダー名 |
|---|---|---|
| `ShaderTypeDefault` | 0 | Default |
| `ShaderTypeDefaultAlt` | 1 | Default (alt) |
| `ShaderTypeEnvironmentMap` | 2 | EnvironmentMap |
| `ShaderTypeGlowShader` | 3 | GlowShader |
| `ShaderTypeHeightmap` | 4 | Heightmap |
| `ShaderTypeMultiLayerParallax` | 10 | MultiLayerParallax |
| `ShaderTypeParallaxOcc` | 11 | ParallaxOcc |
| `ShaderTypeMultiLayerParallaxOcc` | 15 | MultiLayerParallaxOcc |
| `ShaderTypeEnvironmentMapFO3` | 17 | EnvironmentMap (FO3 variant) |
| `ShaderTypeSkinTint` | 31 | SkinTint |
| `ShaderTypeHairTint` | 32 | HairTint |
| `ShaderTypeParallaxOccInner` | 33 | ParallaxOccInner |
| `ShaderTypeTallGrass` | 34 | TallGrassShader |

### 2. ShaderType 別のマテリアルビルダー

`BuildMaterial` が ShaderType でスイッチし、各タイプ専用のメソッドにディスパッチ:

#### Default (type 0, 1)
- Standard PBR: Roughness=0.6, Metallic=0.0
- ShaderFlags ビット 0 (Specular) で SpecularMode を制御
- ShaderFlags ビット 7 で Metallic=0.3, Roughness=0.5
- 共通テクスチャ（Slot 0-1, 4-5）+ Emission + Heightmap + Detail

#### EnvironmentMap (type 2, 17)
- Metallic=0.8, Roughness=0.3 で反射面を強調
- Slot 4 (Environment) を MetallicTexture に設定
- Slot 5 (EnvironmentMask) を RoughnessTexture に設定
- 反射の強い金属・ガラス表現

#### GlowShader (type 3)
- `DisableAmbientLight = true` で環境光を無効化
- AlbedoTexture を EmissionTexture としても使用（全面発光）
- Slot 2 (Glow) があれば EmissionTexture に上書き
- SpecularMode.Disabled

#### SkinTint (type 31)
- Subsurface Scattering を有効化（Skin モード, Strength=0.3）
- `VertexColorUseAsAlbedo = true` で頂点カラーのティント反映
- Slot 2 (Glow) を SubsurfScatterTransmittanceColor のサンプルに使用
- 透過光の色味を近似

#### HairTint (type 32)
- `VertexColorUseAsAlbedo = true`（髪の色を頂点カラーで制御）
- Roughness=0.7 でややマット
- Subsurface はオフ

#### TallGrass (type 34)
- Transparency = AlphaScissor (threshold=0.3)
- Roughness=0.9, Metallic=0.0, SpecularMode.Disabled

#### MultiLayerParallax (type 10, 11, 15, 33)
- Default 設定 + Heightmap + DetailAlbedo（Slot 6）
- ParallaxScale / ParallaxMaxPasses を反映

#### Heightmap (type 4)
- Default 設定 + Heightmap（Parallax 有効）

### 3. 共通テクスチャ適用の改善

`ApplyCommonTextures` メソッドを導入。すべてのタイプで以下のルールでテクスチャをマッピング:

| Slot | FO3 用途 | Redot マッピング |
|---|---|---|
| 0 | Diffuse | AlbedoTexture |
| 1 | Normal + Gloss | NormalTexture (gloss は破棄) |
| 4 | Environment | MetallicTexture (Red チャンネル) |
| 5 | Environment Mask | RoughnessTexture (Red チャンネル) |

**ノーマルマップアルファの Gloss → Roughness 直接マッピングを削除**（Gloss は Roughness と逆方向の値のため）

### 4. NIFBlockResolver との整合性

NIFBlockResolver は既に以下の ShaderType をセットしていた:
- BSShaderPPLightingProperty → NIF から読み取った生値
- BSShaderNoLightingProperty → 生値（主に type 3 の GlowShader）
- NiTexturingProperty / TileShaderProperty → ハードコード 1

すべてのタイプが正しくスイッチされる。

## 残課題

- **カスタムシェーダー**: StandardMaterial3D の制限により、FO3 の正確な Gloss→Roughness 変換（反転）にはカスタムシェーダーが必要
- **Wing (type 29)**: 未実装（半透明の羽根）
- **SnowShader (type 14)**: 未実装（積雪エフェクト）
- **LODLandscape / LODBuilding (type 6, 7, 35)**: Default 扱い。LOD システム導入時に再検討
- **ZBufferWrite (type 5)**: 未実装（特殊深度処理）
- **SkinTint の透過光色**: Slot 2 の (0,0) ピクセルで近似しているが、本来はテクスチャ全体からサンプリングすべき
- **DisableAmbientLight と EnvironmentMap**: GlowShader の DisableAmbientLight が正しく機能するか確認が必要
