# feat_skinning-parser: NiSkinInstance/NiSkinData パース基盤

## 背景
キャラクター NPC/クリーチャーの NIF メッシュは NiSkinInstance によるスキニングで
変形する。スキニング未対応だとスキニングブロックが無視され、NIF パースは
Static と同様に扱われていた。

## 変更内容

### NIFBlockResolver.cs
- `Node` クラスに `SkinInstanceIndex` フィールドを追加（デフォルト -1）
- `ParseNode()` で NiTriShape/NiTriStrips の Skin Instance Ref（int32）を
  読み飛ばしから保存に変更

### NIFReader.cs
- `SkinDataStore SkinData` フィールドを追加
- `ParseSkinningBlocks()` で全ブロックを走査:
  - NiSkinInstance → `ParseSkinInstance()`
  - BSDismemberSkinInstance → `ParseSkinInstance()`
  - NiSkinData → `ParseSkinData()`

### NIFSkinningData.cs（新規）
- **NiTransformData**: Rotation (Basis) + Translation + Scale
- **BoneVertWeight**: VertexIndex (ushort) + Weight (float) = 6 bytes
- **BoneDataEntry**: SkinTransform + BoundingSphere + VertexWeights[]
- **SkinDataInfo**: SkinTransform + Bones[] (NiSkinData 全体)
- **SkinInstanceInfo**: SkinDataIndex + SkinPartitionIndex + SkeletonRootIndex + BoneIndices[]
- **SkinDataStore**: SkinInstances[int] + SkinDatas[int]
- **NIFSkinningParser**: 静的 ParseSkinInstance / ParseSkinData メソッド

### バイナリレイアウト検証（nifxml 基準、FO3 20.2.0.7）
- NiSkinInstance: Data(4) + SkinPartition(4) + SkeletonRoot(4) + NumBones(4) + Bones[NumBones](4ea)
- NiSkinData: SkinTransform(52) + NumBones(4) + HasVertexWeights(1) + BoneList[NumBones]
  - **注意**: NiSkinData の Skin Partition Ref は until="10.1.0.0" により FO3 では存在しない
- BoneData: SkinTransform(52) + NiBound(16) + NumVertices(2) + BoneVertData[NumVertices](6ea)
- NiTransform: Rotation(36=4*9) + Translation(12=4*3) + Scale(4) = 52 bytes
- NiBound: Center(12=4*3) + Radius(4) = 16 bytes
- BoneVertData: Index(2=ushort) + Weight(4=float) = 6 bytes
